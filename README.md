# Vishal Sharma - Portfolio Website

A modern, responsive portfolio website built with React.js, showcasing my experience as a Backend Engineer and my journey in software development.

## Features

- **Responsive Design**: Fully responsive across all devices
- **Modern UI/UX**: Clean, professional design inspired by modern portfolio standards
- **Smooth Animations**: Engaging user experience with smooth transitions
- **Interactive Components**: Dynamic navigation, tabbed experience section, and more
- **Performance Optimized**: Fast loading and smooth scrolling

## Technologies Used

- React.js 18
- CSS3 with custom properties
- React Icons
- Modern JavaScript (ES6+)

## Sections

1. **Hero**: Eye-catching introduction with call-to-action
2. **About**: Personal background and journey
3. **Experience**: Professional work history at Locus.sh
4. **Skills**: Technical skills with visual progress bars
5. **Education**: Academic background and achievements
6. **Contact**: Multiple ways to get in touch

## Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project directory:
   ```bash
   cd Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

To create an optimized production build:

```bash
npm run build
```

The build folder will contain the optimized files ready for deployment.

## Deployment

This site can be easily deployed to:

- **Vercel**: Connect your GitHub repository for automatic deployments
- **Netlify**: Drag and drop the build folder or connect via Git
- **GitHub Pages**: Use `gh-pages` package for easy deployment
- **AWS S3 + CloudFront**: For scalable hosting

## Customization

To customize the portfolio for your own use:

1. Update personal information in the component files
2. Replace the profile photo in `public/assets/`
3. Modify color scheme in `src/index.css` (CSS variables)
4. Add your projects, experiences, and skills
5. Update social media links in the Contact component

## Color Scheme

The website uses a carefully selected color palette:
